列表

列表格式也很常用，在 Markdown 中，你只需要在文字前面加上 - 就可以了，例如：

  • 文本1
  • 文本2
  • 文本3

如果你希望有序列表，
也可以在文字前面加上 1. 2. 3. 就可以了，例如：

  1. 文本1
  2. 文本2
  3. 文本3

注：-、1. 和文本之间要保留一个字符的空格。
